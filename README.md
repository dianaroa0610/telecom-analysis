# Telecom Customer Analysis

## Objetivo del proyecto
Analizar el comportamiento de clientes de telecomunicaciones utilizando Python y técnicas de análisis de datos.

## Datasets utilizados
- users.csv
- usage.csv
- plans.csv

## Etapas del análisis
1. Limpieza y revisión de datos
2. Tratamiento de valores nulos
3. Estadísticas descriptivas
4. Agrupación de uso por usuario
5. Visualización de distribuciones
6. Identificación de outliers
7. Segmentación de clientes

## Herramientas utilizadas
- Python
- Pandas
- Seaborn
- Matplotlib

## Cómo ejecutar el proyecto
1. Descargar el notebook `.ipynb`
2. Abrir en Jupyter Notebook o Google Colab
3. Ejecutar las celdas en orden

## Resultados generales
Se identificaron patrones de uso entre clientes, diferencias entre planes y segmentos de usuarios según edad y comportamiento de uso.
